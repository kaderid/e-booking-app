# 🌍 E-Booking Project

## 🧾 Description
E-Booking est une application complète de **réservation en ligne** développée avec :

- 🖥️ **Backend :** Spring Boot (Java)
- 💻 **Frontend :** Angular
- 🐘 **Base de données :** PostgreSQL (via Docker)
- ⚙️ **Outils :** Docker Compose, pgAdmin
